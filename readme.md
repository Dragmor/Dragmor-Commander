# DragmorCommander

**DragmorCommander** — это графический файловый менеджер, разработанный на Python с использованием библиотеки `tkinter`. Приложение предоставляет удобный интерфейс для управления файлами и папками, поддерживает работу с несколькими панелями и вкладками, а также позволяет подключать сторонние инструменты.

## Основные возможности

### Файловый менеджер

- **Две панели** для одновременной работы с разными каталогами.
- **Многочисленные вкладки** в каждой панели для работы с несколькими каталогами одновременно.
- **Быстрая навигация** по файловой системе:
  - Переход в родительский каталог.
  - Выбор диска из списка доступных.
  - Переход к первому или последнему элементу в списке.
- **Операции с файлами и папками**:
  - Копирование, перемещение, удаление.
  - Вырезание и вставка.
  - Создание новых папок.
  - Переименование файлов и папок (в том числе групповое).
- **Поиск файлов** по имени с помощью строки поиска.
- **Отображение иконок** для файлов и папок.
- **Контекстное меню** для работы с файлами (ПКМ):
  - Открыть.
  - Открыть через...
  - Копировать.
  - Вырезать.
  - Вставить.
  - Переименовать.
  - Создать каталог.
  - Удалить.

### Работа с вкладками

- **Добавление новых вкладок** с помощью:
  - `Ctrl+Enter` — открывает выбранный каталог в новой вкладке.
  - Средняя кнопка мыши на пустом месте панели вкладок.
- **Закрытие вкладок**:
  - Средняя кнопка мыши по заголовку вкладки.
- **Переключение между вкладками**:
  - `Ctrl+Стрелка влево/вправо` — переключение между вкладками в активной панели.

### Горячие клавиши

- `Ctrl+C`, `Ctrl+V`, `Ctrl+X` — копирование, вставка, вырезание.
- `Delete` — удаление выбранных элементов.
- `Ctrl+A` — выделение всех элементов.
- `Backspace` — переход в родительский каталог.
- `Стрелка влево/вправо` — переключение между панелями.
- `Ctrl+Enter` — открытие выбранного каталога в новой вкладке.

### Поддержка инструментов

- **Подключаемые модули** инструментов через папку `modules`.
- **Запуск инструментов** в отдельных процессах.
- **Передача параметров** инструментам (например, путь к выбранным файлам).
- **Доступ к инструментам** через меню **"Инструменты"**.

### Кастомизация интерфейса

- Возможность изменения иконок для файлов и папок.
- **Планируется** добавление настроек для изменения цветов и шрифтов.
